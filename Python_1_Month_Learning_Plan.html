üêç 30-Day Python Learning Roadmap for Beginners

Author: Pavithran Chan

This roadmap is designed to take you from zero to building small projects in just one month. Each day includes a specific topic to focus on and check off.

üìÖ Week 1: The Fundamentals (Building Blocks)

Goal: Run your first script and understand how to store and manipulate basic data.

Day 1: Setup & First Steps

  • ☐ Install Python: Download from python.org (ensure "Add to PATH" is checked).
  • ☐ Install VS Code: Setup your editor with the Python Extension.
  • ☐ Hello World: Write print("Hello, World!") in a file named main.py and run it.
  • ☐ The REPL: Open your terminal, type python, and run simple math (2 + 2).

Day 2: Variables & Data Types

  • ☐ Variables: Learn how to assign values (x = 5, name = "Alice").
  • ☐ Strings: Basics ("text"), string combination ("hello " + "world").
  • ☐ Integers & Floats: Whole numbers vs decimals.
  • ☐ Booleans: True and False.

Day 3: Basic Input & Output

  • ☐ User Input: Use input("What is your name? ").
  • ☐ Type Conversion: Convert strings to numbers (int("5"), str(10)).
  • ☐ F-Strings: Learn specific string formatting (print(f"Hello {name}")).

Day 4: Operators

  • ☐ Arithmetic: +, -, *, /.
  • ☐ Modulo: % (remainder).
  • ☐ Comparison: ==, !=, >, <.
  • ☐ Logical: and, or, not.

Day 5: Control Flow (Making Decisions)

  • ☐ If Statements: if condition:.
  • ☐ Else/Elif: Handling multiple cases.
  • ☐ Indentation: Understand why whitespace matters in Python!

Day 6: Conditionals Practice

  • ☐ Practice: Write a script that checks if a user is old enough to vote (age >= 18).
  • ☐ Mini-Project: Build a simple "Guess the Number" game.

Day 7: Review & Rest

  • ☐ Review: Go back over the code you wrote this week.
  • ☐ Quiz Yourself: Can you explain what a variable is to a friend?

üìÖ Week 2: Logic & Data Structures

Goal: Write smarter code that can handle lists of items and repeat actions.

Day 8: Loops (Iteration)

  • ☐ For Loops: Iterate over a range (for i in range(5):).
  • ☐ While Loops: Run code while a condition is true.

Day 9: Lists (Arrays)

  • ☐ Creating Lists: my_list = [1, 2, 3].
  • ☐ Accessing Items: Indexing (my_list[0]).
  • ☐ Modifying: Changing values, adding (.append()), removing (.pop()).

Day 10: Iterating Lists

  • ☐ Looping Lists: for item in my_list:.
  • ☐ List Methods: len(), sort(), reverse().

Day 11: Dictionaries (Key-Value Pairs)

  • ☐ Basics: person = {"name": "John", "age": 30}.
  • ☐ Accessing: person["name"].
  • ☐ Adding/Editing: person["city"] = "New York".

Day 12: Functions (Reusable Code)

  • ☐ Defining: def my_function():.
  • ☐ Parameters: Passing data in (def greet(name):).
  • ☐ Return Values: Getting data out (return x + y).

Day 13: Scope & Modules

  • ☐ Global vs Local: Where variables "live".
  • ☐ built-in modules: import math, import random.
  • ☐ Practice: Use random.randint() in your guessing game.

Day 14: Review & Mini Project

  • ☐ Project: Build a "To-Do List" app that runs in the console using a list and a loop.

üìÖ Week 3: Advanced Basics

Goal: Handle errors, work with files, and object-oriented concepts.

Day 15: Error Handling

  • ☐ Exceptions: What happens when code crashes?
  • ☐ Try/Except: Catching errors gracefully (try: ... except ValueError:).

Day 16: File Handling (I/O)

  • ☐ Reading: open("file.txt", "r").
  • ☐ Writing: open("file.txt", "w").
  • ☐ Context Manager: Using with open(...) (Best Practice).

Day 17: Sets & Tuples

  • ☐ Tuples: Immutable lists (1, 2).
  • ☐ Sets: Unique items only {1, 2, 3}.

Day 18: Object-Oriented Programming (OOP) Intro

  • ☐ Classes: The blueprint (class Dog:).
  • ☐ Objects: Creating an instance (my_dog = Dog()).

Day 19: Class Methods & Attributes

  • ☐ init: The constructor method.
  • ☐ Self: Understanding self.
  • ☐ Methods: Functions inside classes.

Day 20: Inheritance

  • ☐ Parent/Child: Creating a generic class and specific versions (e.g., Animal -> Cat).

Day 21: Review & Practice

  • ☐ Refactor: Take your "To-Do List" from Week 2 and rewrite it using Classes.

üìÖ Week 4: Real World Skills

Goal: Install external libraries and build a final portfolio project.

Day 22: Pip & Packages

  • ☐ Pip: formatting pip install requests.
  • ☐ PyPI: Browsing the Python Package Index.

Day 23: Virtual Environments

  • ☐ venv: Why we isolate projects.
  • ☐ Create: python -m venv venv.
  • ☐ Activate: Turning it on.

Day 24: Working with APIs (JSON)

  • ☐ JSON: Understanding the data format.
  • ☐ Requests Lib: Fetching data from the web (e.g., a weather API).

Day 25: Dates & Times

  • ☐ Datetime Module: Getting today's date, formatting strings.

Day 26: Debugging Skills

  • ☐ Print Debugging: The old reliable.
  • ☐ VS Code Debugger: Setting breakpoints and stepping through code.

Day 27: Final Project Planning

  • ☐ Idea: Decide on a project (Calculator, Weather App, Quiz Game).
  • ☐ Plan: Write down the features you need.

Day 28: Final Project - Setup

  • ☐ Scaffold: Create files, set up venv, install packages.

Day 29: Final Project - Coding

  • ☐ Code: Write the logic! Focus on getting it working.

Day 30: Final Project - Polish & Celebrate! üéâ

  • ☐ Clean Code: Add comments, remove unused variables.
  • ☐ Readme: Write a small README.md explaining your project.
  • ☐ Celebrate: You are a Python programmer!
